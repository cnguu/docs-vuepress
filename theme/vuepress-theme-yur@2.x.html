                        # Install

                        $ yarn add vuepress-theme-yur

                        Use theme

                        module.exports = {
    theme: 'yur'
}

                        # Basic Information

                        module.exports = {
    themeConfig: {
        lang: 'zh-CN',
        logo: '/logo.png',
        author: 'cnguu',
        authorLink: 'https://github.com/cnguu/',
        locales: {
            title: 'Stars so bright',
            description: 'Writing life with you'
        }
    }
}
                        Parameter Description
                        lang Default language
                        Optional: en zh-CN
                        logo Website logo link
                        author Webmaster name
                        authorLink Webmaster name off-site jump link
                        locales Language localization
                        See details: Multilingual file (opens new window)

                        module.exports = {
    themeConfig: {
        post: {
        	bubbles: {
				color: 'random',
				radius: 15,
				density: 0.3,
				clearOffset: 0.2
			}
        }
    }
}
                        Parameter Description
                        color Bubble color
                        default random
                        radius Bubble radius
                        default: 15
                        density Bubble density
                        the higher the value, the denser the bubble (recommended <= 1)
                        default: 0.3
                        clearOffset Bubble disappearing distance [0-1]
                        the larger the value, the later it disappears
                        default: 0.2

                        # FAQ

                        1. After upgrading 2.x version, the display is still 1.x version?

                          A: The 1.x version uses the pwa plugin by default, and the 2.x version is no longer built-in, which causes the cache to be manually cleared by the user to display the latest.

                          S: Install the pwa extension. If you do not continue to use pwa, you need to write your own js script to automatically clear the old cache.

                        2. Missing index.html after building?

                          S: Create a new file /docs/index.md.

                        3. Abnormal layout of article list?

                          S: Install dependencies yarn add ant-design-vue@1.4.11.
