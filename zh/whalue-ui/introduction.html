        # Whalue UI of Vue

        基于 Vue 实现的个人组件库，开发中

        # 设计思路

        • 通用型组件
          • Icon（图标）
          • Button（按钮）
        • 布局型组件
          • Grid（栅格）
          • Layout（布局）
        • 导航型组件
          • Menu（导航菜单）
          • Dropdown（下拉菜单）
          • Breadcrumb（面包屑）
          • PageHeader（页头）
          • Pagination（分页）
          • Steps（步骤条）
          • Affix（固钉）
        • 数据录入型组件
          • Form（表单）
          • FormModel（表单）
          • Input（输入框）
          • InputNumber（数字输入框）
          • Select（选择器）
          • Radio（单选框）
          • Checkbox（多选框）
          • Switch（开关）
          • TimePicker（时间选择器）
          • DatePicker（日期选择框）
          • Upload（上传）
          • Transfer（穿梭框）
          • TreeSelect（树选择）
          • Slider（滑动输入条）
          • Cascade（级联选择）
          • AutoComplete（自动完成）
          • Mentions（提及）
          • Rate（评分）
        • 数据展示型组件
          • Card（卡片）
          • List（列表）
          • Description（描述列表）
          • Tabs（标签页）
          • Tab（标签）
          • Table（表格）
          • Carousel（轮播图）
          • Collapse（折叠面板）
          • Tree（树形控件）
          • Calendar（日历）
          • Avatar（头像）
          • Badge（徽标数）
          • Comment（评论）
          • Statistic（统计数值）
          • Timeline（时间轴）
          • Popover（气泡卡片）
          • Tooltip（文字提示）
          • Image（图片）
          • Empty（空状态）
        • 反馈型组件
          • Alert（警告提示）
          • Drawer（抽屉）
          • Message（全局提示）
          • Modal（对话框）
          • Notification（通知提醒框）
          • PopConfirm（气泡确认框）
          • Progress（进度条）
          • Result（结果）
          • Skeleton（骨架屏）
          • Spin（加载中）
        • 其它组件
          • LocaleProvider（国际化）
          • ConfigProvider（全局化配置）
          • Divider（分割线）
          • Anchor（锚点）
          • BackTop（回到顶部）
